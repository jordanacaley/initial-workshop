HTML Challenge I

Iteration 1 - Inline vs block elements

Write a paragraph containing some span or em tags of different colors. Spans are inline elements, meaning that their default display property value is inline. Those elements will follow the horizonal flow of a document. Inline elements go next to another inline element so you won't have any inline break until you decide to insert one (like here with a br tag).
Images are inline elements too! That's a simple but super important point ot remember! Feel free to dive deeper with the MDN doc.
